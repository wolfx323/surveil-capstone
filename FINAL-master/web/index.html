WEB COMPONENT

The final web component to this project took the form of a splash page with a somewhat experimental, scrolling stream of consciousness-type effect. The user scrolls through, reading along with the script as it poses questions about data privacy and the user's lack of knowledge or awareness about topics relating to data privacy. The resulting portion explains what Surveil is, and where a person may do to get a copy.

To give users more context as to the project's intentions, a simple "WHY?" link prompts users to click through the main splash page to view a more front-facing version of this final Capstone page. The final site was hosted using Cargo Sites II, a modular, sleek, and interactive web hosting platform.
